Account Registration

If the user wants to create an account they need to press the sign up button in the top right of the home page.
Then the user will be redirected to the sign up page.There, in order to accomplish the registration they need to have an active email account. They will also be asked to add a password. This will be the data they will need to add every time they want to login in case they didn't select the remember me option.
